# Contact Management Dashboard

A responsive single-page Angular application built for customer service representatives to view contacts and their associated email addresses.

##  Features
- **Contacts Dashboard:** Displays a complete list of contacts fetched from a mock backend.
- **Detailed View:** Shows contact information along with associated email addresses via nested API endpoints.
- **Real-Time Search:** Allows filtering contacts dynamically by name or role.
- **Skeleton Loader:** Enhanced UI experience while data is fetching.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views without using 3rd party UI component libraries.

##  Tech Stack & Patterns
- **Framework:** Angular (Standalone Components)
- **Language:** TypeScript
- **Styling:** Custom SCSS
- **HTTP Client:** RxJS Observables
- **Testing:** Vitest / Jasmine Unit Testing
